## Descrição

Este repositório contém três implementações de estruturas de grafos, cada uma baseada em um modelo clássico de representação. Todas as versões incluem um menu interativo no terminal para testar as operações.

### Representações implementadas:

- `ListaArestas.py`: Grafo representado por **Lista de Arestas**
- `ListaAdjacencia.py`: Grafo representado por **Lista de Adjacência**
- `MatrizAdjacencia.py`: Grafo representado por **Matriz de Adjacência**

## Funcionalidades disponíveis

Cada implementação permite:

- Inserir e remover vértices
- Inserir e remover arestas (direcionadas ou não)
- Verificar existência de aresta entre dois vértices
- Listar vizinhos de um vértice
- Verificar se um percurso é válido
- Calcular grau de entrada, saída e total de cada vértice
- Exibir o grafo completo

## Como executar

Certifique-se de ter o **Python 3** instalado. Para rodar qualquer versão, use o terminal:

```bash
python lista_arestas.py
python lista_adjacencia.py
python matriz_adjacencia.py
